how to push the code in github:-
1. git init 
2. git add .
3. git commit -m "commit message"
4. git branch -M main
5. git remote add origin <github link>
6. git push -u origin main


if you have given the wrong github link erlier then do these steps :-
1. git remote remove origin
2.  git remote add origin <github link>
4.  git remote -v    -if after running this two output are coming fetch and push then you have steup the correct url
5. git branch -M main
6. git push -u origin main


for future pushes into the git 
1. git add .  -- after this if you want to check if all the files are included then run this command = git status  after running this command u will be getting all files in green
2. git commit -m "your message"
3. git push


to ignore the file from getting commited 
1. create a .gitignore file 
2. run these commands --
3. echo "node_modules/" >> .gitignore
4. echo "*.log" >> .gitignore


if i have commited the .env file earlier and now i have to remove it in the new commit 
1. create and add that file in the .gitignore folder
2. echo ".env" >> .gitignore [agar multiple files hai tho har file mai ye command likho]
3. for stop tracking :-  git rm --cached .env     [agar multiple files hai tho sabko ek line mai add krdo with an space between them and if used * before that it means ki us extension se khatam hone vali jitni bhi file hai usko exclude krdo]
